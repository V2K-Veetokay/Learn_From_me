# History of C


## Origin

>	Developed at Bell Labs in early 1970s by Dennis Ritchie.  

## Purpose

>	Created to implement the UNIX operating system and for system programming.  

## Key features

>	Low-level memory access, pointers, simple keywords, structured programming, and efficiency.

## Standardization
>	ANSI C standardized in 1989; later ISO updates include C99, C11, C18.
